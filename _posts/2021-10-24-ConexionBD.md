```java
System.out.println("Conexión a Base de Datos");
```
En esta aplicación se pretende realizar una conexión a una base de datos, en este ejemplo a la de una veterinaria.

![image](https://user-images.githubusercontent.com/56097204/138591258-4c506f91-2a51-45a9-912f-2be3fd54c7b0.png)

El usuario deberá incluir los datos para la conexión como el usuario y contraseña, el puerto y la ruta de la base de datos.

![image](https://user-images.githubusercontent.com/56097204/138591411-19784bf5-d5b1-475c-b22f-5541d3faffa5.png)

![image](https://user-images.githubusercontent.com/56097204/138591427-90063851-51e0-4c59-888b-14f719acca7e.png)

![image](https://user-images.githubusercontent.com/56097204/138591441-b29a485d-eb2d-4431-a08b-e70dd71b7952.png)

Cuando la base de datos establece conexión el piloto rojo se torna verde.

![image](https://user-images.githubusercontent.com/56097204/138591499-ed28cbd9-5eb4-4e3e-b305-429802e18a3a.png)
